关于博客

我创建这个博客有两个初衷：

首先，这个博客是我日常学习中记录为文档形式的思考内容的备案，我将其汇聚到这里，方便我在未来能够方便地回顾和查找。

其次，我希望我对于某个问题的认识和视角可以为你提供帮助，因此我将内容公开以促进交流，这也是博客的重要精神之一。

这个博客目前涵盖的内容：

  • 读书笔记：我在日常生活中接触到的书籍（包括课程书籍、课外书籍在内）中内容的简要总结。这些笔记以 Latex 语言编写。
  • 书后习题：在我阅读到的书籍中，各种形式习题的个人解答。这些书后习题解答以 Markdown 语言编写。
  • 论文笔记：在我阅读的论文中，关于这个论文创新内容的提取和论文工作的总结。这些论文笔记以 Markdown 语言编写。

博客的许多信息都来自于我在 Github 上维护的以下三个仓库：

  • 读书笔记仓库
  • 习题解答仓库
  • 论文阅读仓库

对于读书笔记仓库，其内容会发送到博客上当我已经阅读完整本书，并且只会上传导出的 PDF 文件到博客。

对于习题解答仓库，其内容会发送到博客上当我做完了一章的习题或我做完了全部的习题。对于前者，我会将这一章的习题内容汇总导出为单个 HTML 文件进行上传。对于后者，我会将整本书所有习题内容汇总导出为单个 PDF 文件进行上传。

对于论文阅读仓库，其内容会发送到博客上当我已经阅读完这篇论文，并且会上传导出的 HTML 文件和 PDF 文件。

三个仓库的维护与博客是异步的，读书笔记与习题解答仓库的更新普遍快于博客，而论文阅读仓库的更新普遍慢于博客。
